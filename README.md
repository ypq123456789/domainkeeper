# DomainKeeper

DomainKeeper 是一款闭源的域名管理软件。

## 在线演示

- 演示地址：<https://ym.bacon123.eu.org/>

## 使用入口

- 注册入口：<https://ym.bacon123.eu.org/register>
- 登录入口：<https://ym.bacon123.eu.org/login>

注册成功后，每位用户都会拥有自己的专属访问地址：

- 前台：`/{用户名}`
- 后台：`/{用户名}/admin`
